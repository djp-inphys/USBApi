Detailed Description

The Header file for the Vt::CVtABDiff class, used by the ceph calibration system.

The current verson of the pano and ceph system calculate the difference between adjacent tiles. This file contains the class which calculates the offset difference between tiles A&B. Hence, this class is only useful in the cephelometric system.
