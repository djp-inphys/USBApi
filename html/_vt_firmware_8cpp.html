Detailed Description

The file contains the main firmware interface object.

This VtFirmware is the main interface class between the driver and the low level library function the windriver interface library lib.lin. The firmware class is responsible for downloading the firmware, loading the driver and attaching registering the driver with the windriver sub-system.
