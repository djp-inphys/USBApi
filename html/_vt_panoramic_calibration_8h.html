Detailed Description

This file holds all the routines require for pano and ceph calibration.

Currently the pano and ceph systems use a flat field removal calibration approach based on gain and offset calculation.

The calibration class contains the routines for a calibration run, and for calibration application.

See also
Vt::CVtPanoramicCalibration for more details.

This is class which implements the panoramic and cephelometric calibration functions.

The functions may be thought of as consisting of two distince groups those realated to the calculation of the calibration data and those related to the applicaton of the calibration process to acquired images. The former set consist of routines;-

  1. for the acquisition of source data required for calibration
  2. for calculation and storage of the calibration data.
