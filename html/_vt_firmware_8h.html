Detailed Description

The file contains the main firmware interface object.

This VtFirmware is the main interface class between the driver and the low level library function the windriver interface library lib.lib. The firmware class is responsible for downloading the firmware, loading the driver and attaching registering the driver with the windriver sub-system.

The libraries for accessing the low level hardware are layered. At the bottom layer is the lib library. The lib library contained all the windriver used level functions. Above this is the ez_lib library. This contains what is effectively the hardware abstraction layer for data acquisition. This Library has two main classes the USB driver class Vt::CVtUSBDriver and class that wraps all the functionality todo with the firmware. This file contain the class that related to the firmware Vt::CVtFirmware.
